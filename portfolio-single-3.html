ecogreens

Project Description

Ecogreens is an innovative application designed to assist users in calculating the electricity consumption in their homes. Featuring a user-friendly interface and built with cutting-edge technologies such as Jetpack Compose and Kotlin, this application provides users with the convenience to monitor and analyze their energy usage efficiently. One notable feature of "Ecogreens" is its integration with Firebase as a database, ensuring the security and reliability of user data and enabling real-time data access.

Project Date Desember 10th, 2023
Role Android, UI/UX, Machine Learning

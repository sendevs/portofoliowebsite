Project Description

As a dedicated UI/UX designer, I take pride in my latest project the creation of a user-friendly and visually appealing middle school mathematics e-learning application. Utilizing the powerful design tool Figma, I've crafted an interactive and engaging user interface to enhance the learning experience for high school students. The application is thoughtfully designed to focus on key mathematical concepts, offering a 3D geometric feature for spatial understanding, an integrated quiz feature for interactive assessments, and a rating system to gather valuable feedback. This e-learning platform aims to make the study of mathematics not only educational but also enjoyable for students, fostering a positive and effective learning environment
